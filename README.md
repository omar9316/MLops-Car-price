# MLops-Car-price (rebuild)

But : prédire le prix d'une voiture d'occasion — projet reconstruit pas à pas avec une architecture MLOps claire.

## Structure
- src/ → code de l’application (FastAPI, pipelines, etc.)
- notebooks/ → exploration et modèles
- data/ → jeux de données
- models/ → modèles entraînés
- infra/ → Docker, Prometheus, Grafana
- tests/ → tests unitaires
